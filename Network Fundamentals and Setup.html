# Basic Network Fundamentals > **Understanding the Foundation - How the Internet Works** Before we write a single line of code, let's understand the magic that makes it all possible. ## The Big Picture: Client-Server Architecture Every time you: - Send a WhatsApp message - Watch a YouTube video - Submit a form on a website You're participating in a **client-server interaction**. Your device (client) talks to powerful computers (servers) somewhere in the world. **Key Concepts to Explore:** - What happens when you type google.com in your browser? - How does your ISP route your request? - What is BDIX and why does it make your internet faster for local content? - Client vs Server: Who does what? ## Your Learning Mission **Research and understand:** 1. **IP Addresses**: What they are and how they work 2. **DNS**: The internet's phonebook 3. **HTTP/HTTPS**: How web communication happens 4. **Ports**: Different doors for different services 5. **ISP and Routing**: The journey of your data packets **Recommended Resources:** - [How the Internet Works - MDN](https://developer.mozilla.org/en-US/docs/Learn/Common_questions/How_does_the_Internet_work) - [Client-Server Architecture Explained](https://www.youtube.com/watch?v=L5BlpPU_muY) - Watch: "What happens when you type google.com" videos on YouTube ## Set Up Your Development Environment ### Install Node.js Since our entire program revolves around Node.js, let's get it installed: **For Windows:** 1. ### Create Your First Server Let's write your first Node.js server! Create a new folder called backend-learning and follow these steps: 
bash
# Create project directory
mkdir backend-learning
cd backend-learning

# Initialize npm project
npm init -y

# Create your first server file
Create a file called hello-server.js: 
javascript
// hello-server.js
const http = require("http");
const myName = "Your Name";

const server = http.createServer((req, res) => {
  // Set response headers
  res.writeHead(200, { "Content-Type": "text/plain" });

  // Send response
  res.end(`Hello, I'm ${myName}! Saying Hi from my first Node.js server!\n`);
});

const PORT = 3000;

server.listen(PORT, () => {
  console.log(`Server running at http://localhost:${PORT}/`);
  console.log("Press Ctrl+C to stop the server");
});
**Run your server:** Open your terminal and run the following command:
